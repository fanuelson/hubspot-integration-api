## Prerequisites

- Docker
- Docker compose 2

## Run

1. Adicionar ```CLIENT_ID``` e ```CLIENT_SECRET``` fornecidos pelo hubspot ao application.yml 
2. Executar a aplicação:
```
$ docker compose up -d
```

Access:
http://localhost:8080/swagger-ui/index.html
